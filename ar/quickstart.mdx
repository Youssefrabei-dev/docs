---
title: "البدء السريع"
description: "ابدأ بدليل يقدّم أسرع طريق لتحقيق نتيجة ناجحة"
---

اشرح كيف يبدأ شخص ما باستخدام منتجك. ما هو أول شيء يحتاج إلى القيام به؟ هل هناك أي متطلبات مسبقة؟

ينبغي أن ينقل البدء السريع المستخدم من نقطة الصفر إلى استخدام منتجك. سيحقق إنجازًا سريعًا ويحصل على فكرة عمّا يمكنه إنجازه.

## المتطلبات المسبقة

قبل أن تبدأ، يجب أن يتوفر لديك:

- المتطلب الأول (على سبيل المثال، Node.js 18 أو أحدث، حساب مجاني، مفتاح API)
- المتطلب الثاني (على سبيل المثال، جهاز متوافق، متصفح متوافق، نظام تشغيل متوافق)

## البدء

<Steps>
  <Step title="التثبيت">
    اشرح كيفية تثبيت منتجك أو التسجيل فيه.

    ```bash
    npm install your-package
    ```
  </Step>
  <Step title="الإعداد">
    اشرح أي إعداد أو تهيئة مطلوبة قبل الاستخدام الأول.

    ```bash
    your-cli init
    ```
  </Step>
  <Step title="التشغيل">
    أظهر أول شيء يقوم به المستخدم لرؤية المنتج يعمل.

    ```bash
    your-cli start
    ```
  </Step>
</Steps>

<Tip>
  امنح الأشخاص وسيلة للحصول على المساعدة. يمكن أن يكون ذلك رابطًا لصفحة دعم، أو محادثة مع وكيل دعم العملاء، أو منتدى لمنتجك.

  مثال: تحتاج إلى مساعدة؟ تواصل معنا عبر [support@yourcompany.com](mailto:support@yourcompany.com).
</Tip>

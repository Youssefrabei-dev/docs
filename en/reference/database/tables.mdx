---
title: "Tables index"
description: "PostgreSQL tables organized by domain."
---

All tables are defined in `SawaApp/api/db/schema/`.

## Identity and auth

| Table | Purpose |
|-------|---------|
| `users` | User profile + Better Auth identity |
| `genders` | Gender lookup |
| `session` | Better Auth sessions |
| `account` | Passwords and OAuth accounts |
| `verification` | Verification tokens |

## Discovery

| Table | Purpose |
|-------|---------|
| `categories` | Place/activity categories |
| `category_types` | Category type lookup |
| `places` | Venues |
| `activities` | Things to do |
| `plactivities` | Place ↔ activity links |
| `places_media` | Images per place (S3 keys) |
| `media_types` | Media type lookup |
| `places_reviews` | Text reviews |
| `places_reactions` | User reactions |
| `reactions_types` | Reaction type lookup |

## Social

| Table | Purpose |
|-------|---------|
| `companionships` | Friend requests |
| `companionships_status` | Status lookup (pending, accepted, etc.) |
| `companionships_status_changelog` | Status change history |

## Events

| Table | Purpose |
|-------|---------|
| `events` | Event container |
| `event_steps` | Itinerary steps |
| `event_attendees` | RSVP and invites |
| `event_step_votes` | Votes on steps |
| `event_step_droppers` | Users who dropped a step |
| `event_step_users` | Step assignments |
| `event_step_comments` | Comments on steps |
